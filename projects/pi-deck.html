Pi-Deck

The pi-deck is a raspberry-pi based macro tool inspired by the elgato stream deck that can be used wirelessly and is configured through the webbrowser.

The configuration page is based on django and stores the pi-deck's commands inside a sqlite3 database.
This database then gets accessed through the PyQt5 based GUI program.
When now pressing one of the twelfe Buttons in the GUI, the command associated with it will be send to the PC that's meant to be controlled with the pi-deck.
This PC wil then interpret the command and do one of four things:

  • Open a program or file
  • Open a webpage
  • Type a string
  • Press a certain key combination
